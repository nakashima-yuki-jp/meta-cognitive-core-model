# Meta-Cognitive Core Model (MCM)

Hub repository for the Meta-Cognitive Core Model (MCM) series: Zenodo DOIs, cross-site links, and unified metadata.

## 概要
本リポジトリは **Meta-Cognitive Core Model (MCM)** シリーズの公開ハブです。Zenodo上の本文・補助資料・参照データのDOIを集約し、PsyArXiv/GitHub等との相互リンク、およびメタデータ（JSON等）の統合管理を行います。

## 非診断・非タイプ論（重要）
- 本シリーズは、知性・認知活動の「運用」を記述するための参照モデルです。
- 医療・心理臨床の診断ツールではありません。
- 性格タイプ分類や、個人の属性判定（採用・人事評価・医療判断等）に用いることは意図していません。

Non-diagnostic; not a personality typing scheme.
